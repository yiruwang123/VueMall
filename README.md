## 如有雷同，纯属巧合



## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```



### 目录划分	

##### assets->资源

##### common->公共js

##### views->大的视图

##### store->vuex

##### router->vuerouter

##### network->网络请求

##### components->普通组件->(common->与业务无关,可抽离;content->与业务有关)
